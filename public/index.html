Settings

Backup destination

Default folder where new project backups are stored. Each project keeps a memory of where it was last backed up — re-running Backup updates that same folder rather than creating a duplicate.

App logo

Replaces the brand mark in the header and the running window/taskbar icon — Coding Drives restarts automatically so the new icon is applied everywhere it's loaded. Recommended: 256×256 square PNG or ICO (transparent background); JPG / WEBP also work, and SVG updates the in-app header only.

Statuses

Rename a status or pick a custom hex color. Color applies to status pills, filter chips, and KPIs.

Scan folders

Each folder listed here is scanned for project subfolders.

Excluded folder names

Folders matching these names are skipped during scanning. Type a folder name (e.g., Archive) or click Browse to pick one — the folder's name will be added to the exclude list.
